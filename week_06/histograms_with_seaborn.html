Histograms to visualize a distribution¶

In [8]:
sns.histplot(data = penguins, x='flipper_length_mm')
plt.show()

Use fewer bins to have coarser resolution of the distribution.

In [9]:
sns.histplot(data=penguins, x='flipper_length_mm', bins=4)
plt.show()

Use more bins, we will have fewer observations per bin.

In [10]:
sns.histplot(data=penguins, x='flipper_length_mm', bins=85)
plt.show()

Graphically explore grouping our data different ways¶

Use the sns.catplot() FIGURE level function to create visualizations and apply them to a FIGURE object.

Use the FIGURE level functions to create subplots - FACETS - in one line of code. Do this when we want the FACETS to represent different unique values of variables.

In [16]:
sns.catplot(data=penguins, x='flipper_cat', col='sex', kind='count')

plt.show()

We can also use the COLOR, which Seaborn names as the hue to "break up" variables.

In [17]:
sns.catplot(data=penguins, x='flipper_cat', hue='sex', kind='count')

plt.show()

How does the data "break up" by island?

In [19]:
sns.catplot(data=penguins, x='flipper_cat', hue='sex', col='island',
          kind='count')

plt.show()

Further FACETS associated with the ROW based on the species.

In [20]:
sns.catplot(data=penguins, x='flipper_cat', hue='sex', 
            col='island', row='species',
           kind='count')

plt.show()
